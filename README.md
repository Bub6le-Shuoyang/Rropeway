# Rropeway Desktop

Rropeway 是一个离线的游戏对话脚本编辑器，基于 Electron，面向 Windows 桌面使用。

## 开发启动

```powershell
npm install
npm start
```

## 快捷键

- `Ctrl + N`：新建项目
- `Ctrl + O`：打开 `.scriptroom` 项目
- `Ctrl + S`：保存项目

## 构建 Windows 安装包

```powershell
npm run dist
```

安装包会输出到 `dist/`，支持自定义安装目录并创建桌面快捷方式。

## 项目文件

`.scriptroom` 是经过版本标记的 JSON 项目文件，保存章节、场景、对白块和素材引用。导入的图片/音频会复制到项目旁的 `assets/` 目录中，项目内只保存相对路径，移动整个项目目录后仍可使用。

保存时会先写入临时文件，并在已有项目旁保留 `.backup` 备份。打开项目时会进行格式补齐，避免旧版本或字段缺失导致编辑器崩溃。

## 当前能力

- 章节、场景、对白、旁白、玩家选择分支
- 场景新增、重命名和切换
- 选择角色后添加对白，避免产生无角色内容
- 可增删的多个自定义状态标签
- 对白分段与分段主视角角色
- 左侧手柄拖拽排序与单条内容删除
- 撤销/重做和未保存状态
- 关闭窗口前保存/不保存/取消
- 临时草稿自动保存与恢复
- 图片素材绑定为场景背景或对白立绘
- 本地项目保存和素材导入
